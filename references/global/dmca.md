# DMCA (Digital Millennium Copyright Act, US) — v2.3 (2026-05-25 갱신)

## 트리거
`--global=dmca` 또는 미국 사용자 + UGC (호스팅·검색·캐싱·링킹 제공)

## 핵심 조항 — 17 U.S.C. §512 (Safe Harbor)

### §512(a) 단순 전송 (Conduit)
- ISP·CDN — 콘텐츠를 단순 전송·라우팅. 내용 인지·통제 ✗
- 면책 요건: 자동·일시적 저장, 사용자 선택, 비변경

### §512(b) 시스템 캐싱
- 프록시·캐싱 서버
- 면책: 비변경·접근 조건 준수·갱신 룰 준수

### §512(c) 호스팅 (UGC 플랫폼 핵심)
- **YouTube·Patreon·Cre8orClub 등 적용**
- 면책 요건 5건:
  1. **실제 인지 또는 red-flag 인지 ✗** (구체적 게시물 단위)
  2. **인지 즉시 신속 제거** ("expeditiously")
  3. **직접 금전적 이익 ✗** (침해 콘텐츠와 직결되어 발생하는 이익)
  4. **DMCA Agent 등록** (USCO Designated Agent Directory, 갱신 의무 — **3년마다**)
  5. **Repeat Infringer Policy 채택·공표·합리적 시행**

### §512(d) 정보검색 도구
- 검색엔진·링킹 도구

### §512(g) Counter-Notice (이의제기)
- 호스팅 제공자가 콘텐츠 제거 시 게시자에 통지
- 게시자 Counter-Notice 제출 시 — 호스팅 사업자는 **10~14영업일** 후 복원
- 권리자가 그 사이에 소송 제기 시 복원 보류

## Notice & Takedown 절차

### 권리자 Notice 필수 6요소 (§512(c)(3))
1. **권리자 또는 대리인 서명** (전자서명 OK)
2. **침해 저작물 식별** (특정 작품·시리즈 명시)
3. **침해 자료 식별 + 위치 URL** (제거 위해 충분한 정보)
4. **권리자 연락처** (성명·주소·전화·이메일)
5. **선의 진술** ("a good faith belief that use is not authorized")
6. **정확성 진술 (위증죄 처벌 하)** ("under penalty of perjury, accurate, authorized")

### Counter-Notice 필수 5요소 (§512(g)(3))
1. 게시자 서명
2. 제거된 자료 식별 + 제거 전 위치
3. 위증죄 처벌 하 정확성 진술 + 선의 오류·오인 주장
4. 게시자 연락처 + 관할 동의 (해당 연방지방법원)
5. 송달 동의

## DMCA Agent 등록

- **USCO Designated Agent Directory** 등록 의무 (서비스 제공자)
- 등록 수수료 = **$6** (3년 갱신 / 2024 기준)
- 미등록 = §512(c) 면책 ✗
- 등록 후 처리방침·웹사이트에 Agent 정보 공개 의무

## 2024-2026 동향
- **AI 학습 데이터** 관련 DMCA 분쟁 증가 (NYT v. OpenAI 등) — §512는 학습 단계 적용 명시 ✗
- **YouTube Content ID** = §512 보다 강화된 자체 시스템 (해시 매칭·수익 공유)
- **TikTok·Meta**: 사전 매칭 + 사후 takedown 병행

## 한국법 비교

| 항목 | DMCA §512 | 한국 (저작권법 §103 + 정통망법 §44-2) |
|---|---|---|
| Notice-Takedown | §512(c)(3) 6요소 | 저작권법 §103 + 시행령 §40 양식 |
| Counter-Notice | §512(g) 10~14일 복원 | 정통망법 §44-2 임시조치 30일 |
| Repeat Infringer | 정책 의무 | 저작권법 §133-2 (헤비 업로더) |
| Safe Harbor 면책 | 4유형 명문 | 저작권법 §102 (4유형) |
| Agent 등록 | USCO 의무 | KCC·KCOPA 비공식 |
