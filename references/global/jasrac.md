# JASRAC (일본음악저작권협회) — v2.3 (2026-05-25 갱신)

## 트리거
`--global=jasrac` 또는 일본 음악 사용 (커버·BGM·UGC·라이브)

## 일본 음악 신탁관리 구도

### 2단체 병립 (구조)
- **JASRAC** (1939 설립) — 일본 최대 신탁관리. 작사·작곡·편곡권 + 출판권 일부 신탁
- **NexTone** (2016, e-License + JRC 통합) — JASRAC 독점 견제 신탁. 작사·작곡 신탁 (출판권 ✗)
- **JRRC** — 사적 복제 보상금 관리 (별개 영역)

### 권리 분할
- **저작권 (作詞·作曲)** = JASRAC 또는 NexTone 신탁
- **인접권 (実演家·原盤)** = RIAJ·일본음악원반협회 별도 처리
- → **2단 라이선스** 필수 (작사·작곡 + 원반)

## 사용료 규정 (主要)

### ストリーミング (스트리밍)
- 정액 + 종량 혼합
- 광고 모델 = 매출의 일정 %
- 구독 모델 = 매출 7~10% (상품·서비스별 변동)

### ダウンロード (다운로드)
- 1다운로드당 정액 (가격대별 차등)

### UGC·동영상 사이트
- YouTube·TikTok·Niconico 등 = JASRAC와 포괄계약 (서비스 사업자가 지불)
- 개별 업로더 부담 ✗ (단 NexTone 신탁곡은 별도)

### 라이브·이벤트
- 입장료·정원·연주곡수 기준 차등
- 비영리·소규모 = 별표 감면

## 한일 상호관리협약

- **KOMCA ↔ JASRAC 상호관리협약** — 1996 체결, 갱신 중
- 한국 KOMCA 신탁곡 일본 사용 = JASRAC 경유 라이선스
- 일본 JASRAC 신탁곡 한국 사용 = KOMCA 경유 라이선스
- **NexTone은 KOMCA와 별도 협약** — 곡별 신탁 확인 필수

## 2024-2026 동향

- **JASRAC vs YouTube** 사용료 재협상 (2024-2025) — 광고 매출 기준 비율 상향 조정
- **NexTone 점유율 상승** — K-POP 일본 진출 곡 다수가 NexTone 신탁 → 한국 사업자 양 단체 모두 협상 필요
- **AI 생성 음악** = JASRAC 2025 가이드라인 발표 — "원곡 의거성" 판단 기준 + 신탁 거부권 명시
- **UGC 커버 표준화** — 2025 JASRAC UGC 포괄계약 갱신 (틱톡 포함)

- UGC 약관에 "일본 곡 사용 시 신탁 라이선스 조건 준수" 명시
- AI 커버 = 원곡자 명시 + 신탁 라이선스 + 인접권 별도

## 일본 APPI 연계 (개인정보)

- JASRAC = 음악 권리만. **개인정보는 APPI (個人情報保護法)** 별도
- 일본 유저 가입 = **APPI 2022 개정** 적용:
  - 요배려개인정보 별도 동의
  - 중대 유출 시 PPC(개인정보보호위) + 본인 통지 의무
  - 국외 이전 = 본인 동의 또는 기준적합국 (한국은 2023-12 기준적합국 지정 완료)
