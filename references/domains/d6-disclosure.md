## 2. 리딩 판례·심결 (a2 매핑)

### D6-1. 약관 설명의무 위반
- 대법원 2014다67829 등: **중요 조항**(해지·손해배상·책임제한)은 구체 설명 없이 동의만 받으면 편입 부정
- **앱 함의**: 중요 조항은 체크박스 + 별도 고지 + 팝업 요약. 단순 "약관 전체 동의" 불가

### D6-2. 위치정보법 위반
- 방통위 188사 일괄 제재 (2024-06-12): 과징금·과태료 각 **8.56억**
- 위치정보사업 미신고 / 동의 누락 / 파기 미이행
- **앱 함의**: GPS·Wi-Fi·IP 기반 위치정보 처리 시 **등록제 확인** + D5 연동 동의

### D6-3. 약관 불공정 조항
- 공정위 표준약관·약관심사위 의결례: 일방적 변경·과중한 손해배상·부당 면책 → 무효
- **앱 함의**: "언제든 변경 가능" "어떠한 책임도 지지 않음" 등 문구 금지

## 5. Cre8orClub 특화 이슈

- **통신판매업 신고**: 수익화 시점(굿즈·구독) 신고 의무 발생
- **사업자 정보**: 법인 설립·대표자·주소 확정 후 앱 첫화면·설정 2곳 공개
- **약관 버전 관리**: 개정 시 **30일 전 고지** + 불리한 변경은 동의 재수집
- **외국인 아티스트**: 계약 준거법·관할 명시 (한국법 원칙)

## 6. 도메인 체크리스트

- [ ] 이용약관 공정위 표준약관 기반 작성
- [ ] 사업자정보 앱 내 **2곳 이상** 공개 (설정·약관 내)
- [ ] 전상법 §10 **7대 필수항목** 완비

- [ ] 약관 개정 **30일 전 고지** + 불리 변경 동의 재수집

- [ ] 위치정보사업 신고 (GPS 사용 시)
