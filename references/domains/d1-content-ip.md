## 2. 리딩 판례 (a2 매핑)

### D1-1. BTS 화보집 — 퍼블리시티권 우회로
- **대법원 2020. 3. 26. 자 2019마6525 결정**
- 사건: BTS 무단 화보집 판매
- 판시: 부경법 §2 1호 카목(성과도용) 적용 → 퍼블리시티권 독립 입법 부재 상태에서 **실질 대체**
- **앱 함의**: 판시 근거 = 카목(성과도용) **단독**. 타목(2022-06-08 시행, 성명·초상 식별표지 무단 상업적 사용)은 별도 후행 리스크. 연예인·인플루언서 사진·이름을 프로필·카드·굿즈에 노출하는 UGC 기능 = 카목 즉시·타목 확장 2중 방어 필수. 동의·이용약관 필수

### D1-2. 소리바다 — OSP 방조책임 원형
- 대법원 2007. 12. 14. 선고 2005도872 외
- 판시: 기술적·경제적 관리·통제 가능성 + 침해 인식 → **방조 성립**
- **앱 함의**: 통지 수신 시 즉시 처리 체계 필수. 저작권법 §103 통지절차 = 방조 면책 최저선

### D1-3. OSP URL 특정 요건
- 대법원 2009. 4. 16. 선고 **2008다53812** 전원합의체
- 판시: 침해 게시물 **URL 단위 특정** + 관리·통제 가능성 + 인식 → 방조 책임
- **앱 함의**: Notice & Takedown 폼에 URL 필수 입력 칸. 포괄 신고 = 면책 약화

### D1-4. AI 생성물 저작권 (진행형)
- 미국 Thaler v. Perlmutter (D.C. Cir. 2025): AI 단독 생성물 저작권 등록 불가 확정
- 한국 저작권위원회 2023 지침: "인간의 창작적 기여" 요구
- **2026-01-22 AI 기본법 시행** 후 추가 가이드 예상
- **앱 함의**: AI 생성 프로필·커버이미지 = 저작권 귀속 불명. 약관에 "AI 생성물 라이선스·책임 분배" 명문화 필수

## 5. Cre8orClub 특화 이슈

- **팬아트·2차창작**: 원저작자 동의·공정이용 경계. 약관에 "UGC 업로드 = 비독점 라이선스 플랫폼에 부여" + "원저작자 권리 존중" 2중 조항
- **커버곡·리믹스**: KOMCA(저작권), 음반제작자(저작인접권) 신탁 여부 개별 확인. JASRAC(일본) 오버레이는 R5
- **아티스트 이름·이미지**: 공식 계약·동의서 없이 사용 금지. D1-1 BTS 사건 직접 리스크
- **AI 창작 보조**: AI 기본법 2026-01-22 시행 → 생성 AI 사용 고지·워터마크 의무 대비
