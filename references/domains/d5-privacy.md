## 2. 리딩 판례·심결 (a2 매핑)

### D5-1. SKT 해킹 (2024~2025)
- 개보위 과징금 **1,347억 9,100만원** (2025-08-27)
- 쟁점: 안전조치 미이행·접속기록 미흡
- 행정소송 1심 결과 대기
- **앱 함의**: 접속기록 2년 보존·암호화 저장 필수

### D5-2. 카카오 이모티콘 (2024)
- 개보위 과징금 **151억 4,196만원** + **6만 5,719건** 유출 (2024-05 의결)
- **2026-01-15 1심 패소** — 2심 결과 대기
- 쟁점: 결합용이성·제3자 제공 정의 확대
- **앱 함의**: 해시값·제휴사 데이터 결합 시 재식별 가능성 평가 → DPIA 필수

### D5-3. 홈플러스 1mm 사건 (고전)
- 대법원 2016다200538: 1mm 글씨로 마케팅 동의 받은 경품 응모 이벤트 → **고지 불충분, 동의 무효**
- **앱 함의**: 가입 화면 동의 UI — 필수·선택 분리 + 충분한 가독성. 다크패턴 사전선택 금지

### D5-4. Meta GDPR 과징금
- 아일랜드 DPC **12억 유로** (2023, EU 내 개인정보 역외이전 위반)
- **앱 함의**: GDPR 오버레이 적용 시 SCC·역외이전 평가 필수 (R5)

### D5-5. 우리카드 과징금 134억 5,100만원 (2025-03-27)

## 5. Cre8orClub 특화 이슈

- **15세 이상 정책**: 14세 미만 가입 차단 UI + 생년월일 검증 → 법정대리인 동의 로직 회피
- **팬↔아티스트 DM**: 상대방 동의 + 제공 항목 최소화
- **아티스트 프로필 공개**: 공식 계정은 "공개" 기본, 팬 프로필은 "비공개" 기본
- **해외 아티스트·팬**: GDPR·CCPA·PIPL 오버레이 (R5)
- **접속기록**: 2년 보관 (개보법 시행령 §30 + 고시 2025-10-31)
- **CPO 지정**: 개보법 §31·시행령 §32 — 매출 1,500억 또는 100만 명 이상 처리 시 의무

## 6. 도메인 체크리스트

- [ ] **개인정보처리방침** 11개 필수 기재사항 (개보법 §30)
- [ ] 앱 첫화면·설정·가입 3곳 게시 (2025-04-21 작성지침)
- [ ] 가입 화면 동의 UI — **필수/선택 분리**, 체크박스 사전선택 금지
- [ ] 마케팅 **opt-in** (정통망법 §50) — 야간 광고 21~08시 별도 동의
- [ ] 14세 미만 가입 차단 or 법정대리인 동의
- [ ] 접속기록 **2년 보관**, 암호화 저장
- [ ] 비밀번호·주민번호·생체정보 **저장·전송 암호화**
- [ ] 유출 발생 시 **지체없이** 통지 + 1,000명 이상 개보위 신고
- [ ] 탈퇴 후 즉시 파기 원칙 + 법정 보관 예외 명시
- [ ] CPO 지정·연락처 공개 (처리방침)
- [ ] DPIA (결합용이성 평가) 실시 — 카카오 151억 사건 교훈
