## 2. 리딩 판례·심결 (a2 매핑)

### D3-1. 혐오표현 명예훼손
- 대법원 혐오표현 판례 누적 (2020~): 집단·특정 개인 식별 가능성 + 사실 적시 여부
- **앱 함의**: 커뮤니티 가이드라인 **특정 집단 혐오 예시** 명시 + 신고 → 24시간 검토 SLA

### D3-2. 딥페이크 처벌 강화
- 성폭력처벌법 §14-2 개정 (제작·반포 2020-06-25 / **시청·소지·구입·저장 2024-10-16 시행**)
- 허위영상물 반포 → **7년 이하 징역 또는 5천만원 벌금**
- 편집·합성·가공 단계도 처벌
- **앱 함의**: AI 얼굴 합성 기능 도입 시 **의무고지 + 워터마크** + 사후 신고 채널 필수

### D3-3. 스토킹 처벌법 온라인 적용
- 반복적 DM·팔로우·메시지 = 스토킹 행위 성립 가능
- **앱 함의**: 차단 기능·신고 → 계정 정지 3단 제재 명문화

### D3-4. UGC 사칭·허위정보
- 부경법 §2-1-파 (타인 성명·상호·상표 무단 사용 사칭)
- **앱 함의**: 공식 인증(verified) 배지 + 사칭 신고 전용 폼

### D3-5. 아동·청소년성착취물 (아청법 §11·§17·§25조의2)
- 제작·배포 → 무기 또는 5년 이상 징역 · 소지·시청만으로도 처벌 (§11⑤)
- **통신사업자**: 인지 즉시 삭제·전송방지 조치 의무, 위반 시 3년 이하 징역·3천만원 벌금 (§17)
- **선제적 발견 의무** (§25조의2): 기술적·관리적 조치 의무화 — NCMEC/IWF 해시 DB 매칭 등
- **앱 함의**: 아청물 해시 매칭 + 신고 접수 **즉시(지체없이)** 차단 + 수사기관 신고 프로세스

### D3-6. 임시조치 (정통망법 §44-2·§44-3)
- 권리침해 주장 → 사업자 **30일 접근차단** 가능
- 정보게재자 이의 → **방심위 §44-3 심의** 이행
- **앱 함의**: 임시조치 요청 폼 + 차단 후 게재자 통지 + 이의신청 채널 · 자체 심의 ✗

## 5. Cre8orClub 특화 이슈

- **팬덤 문화**: 지지 집단 결속 강점 ↔ 타 팬덤 공격·저격 리스크 (혐오 성립 가능)
- **아티스트 저격**: 팬→아티스트 명예훼손 / 안티→허위정보 살포
- **사칭·어그로**: 공식 계정 배지 + AI 대응 (딥페이크 합성 콘텐츠 감지)
- **DM 스토킹**: 차단·신고 후 재가입 방지 (디바이스 fingerprint)
- **성적 대상화**: 15세 이상 정책이라도 청보법 교차 적용 (유해매체물 심의)

## 6. 도메인 체크리스트

- [ ] 커뮤니티 가이드라인 **금지 행위 10대 유형** 명문화
- [ ] 3단계 제재 정책 (경고→정지→영구정지) + 각 단계 기준
- [ ] 신고 처리 SLA: **24시간 검토, 72시간 결정**
- [ ] 이의절차 **14일** 보장
- [ ] 딥페이크 감지·차단 기술 (MAU 10만 이상 의무)
- [ ] 사칭 신고 전용 폼 + 공식 배지
- [ ] 차단 우회 방지 (재가입·복수계정)
- [ ] 청소년유해매체물 필터링
- [ ] **아청법 §25조의2 선제적 발견** — 해시 DB 매칭·AI 감지 (아청물 즉시 차단 + 수사기관 신고)
- [ ] **정통망법 §44-2 임시조치** 요청 폼 — 30일 접근차단 + 게재자 통지 + 이의신청 채널
