# α5. 심사관 휴리스틱 (BETA)

⚠️ **데이터 한계**: PIPC 공개 검사결과 DB 접근 제한 + 조사관 인터뷰 부재 → 추정 다수. v2.1 인터뷰 보강 필수.

## 역할
공식 가이드라인에 없지만 실무에서 작동하는 휴리스틱 30개. **신뢰도 "상" 15개 즉시 적용, "중" 12개 + "하" 3개는 BETA 라벨 + `--include-experimental`**.

## 신뢰도 "중" 12개 (BETA)

4. 3자 제공처 "필요시"만 → 제공처 리스트 명세
6. "맞춤형 광고" vs "타게팅" 혼용 → 용어 통일
9. 거부권 "설정에서"만 기술 → 직접 링크
11. 쿠키 "개인정보 아님" 기술 → 쿠키 종류별 분리

16. 국외이전 수신국 보호 무언급 → SCC·GDPR 표기
19. 동의철회 3단계+ 클릭 → 1~2 단계
24. 수탁자 책임한계 미명시 → 계약 의무

## 신뢰도 "하" 3개 (BETA·옵션)

7. 글자 크기 본문 60% 이하 → 85%+
18. 문장 평균 40자 초과 → 짧게
23. "기타" 항목 10%+ → 개별 명시

## v2.0 출력 원칙

```
✅ 신뢰도 "상" 15개: 즉시 출력 + 정상 가중치
⚠️ 신뢰도 "중" 12개: BETA 라벨 + 출력 + 절반 가중치
🔬 신뢰도 "하" 3개: BETA 라벨 + 옵션 출력 (--include-experimental)
```
