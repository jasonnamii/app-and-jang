# α2. 판례 룰엔진

## 역할
한국 법원 약관·개인정보 무효판결 50+건 분석 → 가산/감산 룰 추출 → 무효확률 출력. 케이스노트·로앤비·LBox 검색 + 대법원 판례 공보.

## 무효확률 모델

```
P(무효) = Base(60%) + Σ가산 - Σ감산

Base = 60%  # 한국 법원 약관 무효판결 평균 (2020-2025 추정)

가산:
+15%  명백 불공정 (약관규제법 §8 위반 명시)
+10%  최근 5년 유사 무효판례 존재
+10%  사업자 일방 책임전가
+5%   조항 범위 광범위·모호
+5%   설명의무 미이행

감산:
-15%  명시적 동의 절차
-10%  30일 사전공지 명시
-10%  거부 옵션·환불 절차 명시
-5%   구체 근거법령 인용
-5%   업계 표준 준수
```
