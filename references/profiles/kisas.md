## 🚨 KISAS 특수 맹점 3건 (필수 대응)

### 맹점 ①: 오프라인 캠프 학원법 판정 (🔴 극高)

**쟁점**: 유상 10일 오프라인 몰입 프로그램 = 학원법 §2의2 "학원" 해당 가능.
**온라인 교육은 현재 없음** → 평생교육법 대상 외.

**대응 3가지 경로**:
| 경로 | 절차 | 리스크 |
|------|------|------|
| A. 자체 학원 등록 | 관할 교육지원청 등록 | 미등록 운영=1년 이하 징역·1천만원 이하 벌금(학원법 §22) |
| B. 중앙대 위탁교육 | 수강료·모집 주체를 대학으로 이전 | 계약상 정리 필요·수익 구조 변경 |
| C. 대학 부설 평생교육원 | 중앙대 부설화 | 행정 시간 소요 |

**현재 판정 pending** — 7/27 D-Day 전 관할 안성교육지원청 사전질의 필수.

### 맹점 ②: SBT(Proof Layer) 개인정보 파기권 충돌 (🔴 극高)

**쟁점**: SBT에 개인식별 정보(이름·이메일·에드이디) 온체인 기록 확정.
**법적 충돌**:
- 개보법 §21 (파기 의무) vs 블록체인 불변성
- GDPR Art.17 (Right to Erasure) 정면 충돌
- 개보법 §28-8 (국외이전) — 퍼블릭 체인 사용시 전 세계 노드에 복제

**설계 가드레일 (필수)**:
1. **온체인 = 해시만** — 개인식별정보 직접 기록 ✗
2. **오프체인 DB** — 이름·이메일·에드이디 등 원본은 중앙화 DB에 별도 저장
3. **해시-원본 매칭 키** — 오프체인에만 보관, 파기 요청 시 키 파기 = 사실상 익명화
4. **프라이빗/컨소시엄 체인 검토** — 국외이전 이슈 회피
5. **처리방침 명시** — "SBT 온체인 데이터는 해시값이며 개인식별불가. 원본 파기 시 매칭키 파기로 익명화" 고지

### 맹점 ③: IP 홀딩스 전량 귀속 계약 (🔴 고)

**쟁점**: 크리에이터 창작물 IP를 플랫폼 법인이 아닌 **홀딩스**로 귀속.
표준 UGC 라이선스 템플릿은 플랫폼 법인 기준 → 적용 불가.

**커스텀 계약 필수 요소**:
1. 상대방 표기 = **"KISAS 홀딩스 주식회사"** (플랫폼 법인 ✗)
2. 양도 범위 = 저작재산권 전부 + 2차저작물 작성권 + 퍼블리시티
3. 대가·정산 조항 = 홀딩스 → 크리에이터 직접 지급 구조
4. 계열사 이용권 = 플랫폼·플러스·유학센터 사용 허락
5. 크리에이터 저작인격권 포기 불가 — 행사 제한 합의만

**커스텀 템플릿 생성 대상**: `templates/d1-ugc-license-kisas.md` (v2.2 착수)
