# D2 플랫폼 체크리스트

## OSP 면책 (저작권법 §102·§103)
- [ ] 저작권 침해 신고 접수 창구·이메일 공개
- [ ] 통지·반론통지(counter-notice) 절차 명시
- [ ] 즉시 삭제·차단 조치 SLA
- [ ] 반복침해자 계정 정책 (3-strike 등)

## Notice & Takedown (정통망법 §44-2)
- [ ] 명예훼손·프라이버시 침해 신고 접수
- [ ] 임시조치(30일 이내 결정) 절차
- [ ] 신고자·게시자 통지 의무

## 청보법 (청소년보호법)
- [ ] 청소년유해매체물 접근 차단
- [ ] 광고 필터 (주류·담배·도박)
- [ ] 연령 인증 로직

## N번방법 (전기통신사업법 §22-5)
- [ ] 불법촬영물 즉시 삭제·차단 의무
- [ ] 정기 모니터링 결과 방통위 보고
