# d1-publicity-consent 템플릿 (퍼블리시티권 동의서) — v1.0

## 법적 근거
- **부정경쟁방지법 §2-1-타** (2021-12-07 공포 / **2022-04-20 시행** · 법률 제18548호) — 성명·초상·음성·서명 등 식별표지 무단 사용 금지 명문화
- **민법 §750** (불법행위 손해배상)
- **저작권법 §35-5** (공정 이용)

## 필수 기재 사항 (Cre8orClub UGC·KISAS 학습자료 공통)
1. 권리자 본인 식별 (성명·연락처·생년월일)
2. 사용 대상 식별표지 명시 (사진·영상·음성·서명·아이디 등 — 다중 선택 가능)
3. 사용 목적·범위 (서비스 내 게시·홍보·2차 가공 여부 명시 — 포괄동의 ✗)
4. 사용 기간·지역 (기한 명시 / 무제한 약정 = §6 약관규제법 무효 추정)
5. 대가·보상 조건 (무상 시 명시)
6. 동의 철회 방법·효과 (철회 시 향후 사용 정지 / 기존 발행분 회수 의무 여부)
7. 14세 미만은 법정대리인 동의 별도 (개보법 §22-2 + 청보법 §16)

## 적용 조건 (서비스별 분기)
- **Cre8orClub** — 아티스트 본인 또는 팬 UGC 사용 시 사전 동의 필수. 포괄동의 약관 1회 + 사용 시점 개별 통지 권장
- **KISAS** — 학생 학습자료 영상·사진 사용 시 본인 + 법정대리인 동의 (대부분 미성년)
- **AI 생성** — 본인 사진·목소리로 AI 결과물 생성 시 별도 동의 필수 (부경법 §2-1-타 + AI 기본법 §27)
