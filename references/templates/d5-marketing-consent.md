# d5-marketing-consent 템플릿 (광고성 정보 수신 동의서) — v1.0

## 법적 근거
- **정보통신망법 §50** (영리목적 광고성 정보 전송 제한) — 사전 동의 (opt-in) 원칙
- **정보통신망법 §50-3·§50-4** (수신거부·동의철회 의무)
- **개인정보보호법 §22** (마케팅 별도 동의)
- **전상법 §21-2** (다크패턴 6유형 — 사전선택·반복간섭 금지)

## 필수 구성 (5요소)
1. **별도 동의** (회원가입·필수동의와 묶기 ✗)
2. **수신 채널별 동의** (이메일·SMS·앱푸시 별도)
3. **야간(21~익일 08시) 별도 동의** (§50 ②)
4. **광고 메시지 "(광고)" 표기 의무**
5. **수신거부 링크 본문 포함**

## 문구 예시 (v2.3)

```
[광고성 정보 수신 동의서 — Cre8orClub v1.0]

▶ 회사는 다음 사항에 대해 마케팅·광고 정보를 발송할 수 있습니다.
   1. 신규 아티스트·콘텐츠·이벤트 안내
   2. 멤버십 혜택·쿠폰·할인
   3. 설문·이벤트 참여 안내

▶ 수신 채널별 동의 (별도 선택)
   □ 이메일 수신  □ SMS 수신  □ 앱 푸시 수신

▶ 야간 수신 (21:00 ~ 익일 08:00)
   □ 야간 수신 동의

▶ 동의 거부 시
   서비스 이용에 제한 ✗. 마케팅 정보만 미수신.

▶ 동의 철회
   언제든지 [마이페이지 - 설정 - 알림 설정] 또는
   [unsubscribe@cre8or.club]에서 철회 가능.
   회사는 철회 즉시 발송을 중단합니다.

▶ 광고 메시지 표기
   모든 광고 메시지 제목 또는 본문 시작에 "(광고)" 표기
   본문 하단에 "수신거부" 링크 제공
```

## 적용 조건 (서비스별 분기)
- **Cre8orClub** — 팬 마케팅 푸시 활발 → 채널별·야간 별도 동의 + 다크패턴 6유형 회피 UX 필수
- **KISAS** — 학생 마케팅 ✗ 원칙 (보호자에 한정 + 학원 영업 정보만)
- **글로벌** — GDPR Art.7 + CAN-SPAM Act (미국) + EU PECR (쿠키·전자마케팅)
