# d2-takedown-policy 템플릿 (게시물 차단·임시조치 정책) — v1.0

## 법적 근거
- **정보통신망법 §44-2** (정보의 삭제요청 등) — 권리침해 정보 삭제·임시조치 30일
- **정보통신망법 §44-3** (임의 임시조치) — OSP 자체 판단 임시조치 면책
- **정보통신망법 §44-7** (불법정보 유통 금지)
- **저작권법 §103** (Notice-and-Takedown)
- **전기통신사업법 §22-5** (N번방법 — 매출 10억 또는 DAU 10만+ 의무)
- **성폭력처벌법 §14-2** (불법촬영물 시청·소지 처벌 **2024-10-16 시행**, 법률 제20494호)

## 필수 구성
1. **신고 채널** (24시간 접수 — 웹폼·이메일·앱 내 신고 버튼)
2. **신고 처리 기준 + SLA** (긴급 = 24h·일반 = 7일)
3. **임시조치 → 통지 → 이의제기 → 복원** 절차
4. **반복 침해자 정책** (3-strike 등 단계적 제재)
5. **불법촬영물·아청물 특별 절차** (해시 매칭 + 즉시 차단)
6. **투명성 보고** (연 1회 — §22-5 의무자)

## 문구 예시 (v2.3)

```
[게시물 신고·임시조치 정책 — Cre8orClub v1.0]

제1조 (적용 범위)
회사가 운영하는 서비스 내 모든 게시물·댓글·메시지에 적용합니다.

제2조 (신고 접수)
① 누구든지 다음 사유로 게시물 신고를 할 수 있습니다.
   1. 사생활 침해·명예훼손
   2. 저작권·상표권·퍼블리시티권 침해
   3. 불법촬영물·아동·청소년 성착취물
   4. 음란·폭력·혐오·차별
   5. 사기·도용·계정 탈취
② 신고 채널: [report@cre8or.club] / 앱 내 [⋮] - [신고]

제3조 (처리 기간)
① 긴급(불법촬영물·아청물·즉각 위해) — 인지 즉시 임시조치
② 일반 — 접수 후 7일 이내 검토·통지
③ 정보통신망법 §44-2 임시조치 = 최대 30일

제4조 (임시조치 → 통지 → 이의제기 → 복원)
① 권리침해 주장 신고 → 회사 임시 차단 → 게시자 통지
② 게시자 이의제기 (30일 이내) → 회사 검토 → 복원 또는 유지
③ 신고자·게시자 모두 사법 절차로 분쟁 해결 권고

제5조 (불법촬영물·아청물 특별 절차)
① 해시 매칭 시스템(방심위 DB·NCMEC·IWF) 24시간 가동
② 인지 즉시 차단 + 30일 이내 방심위·수사기관 신고
③ 저장·접근 차단 (저장 자체 시청·소지 처벌 — 성폭력처벌법 §14-2)

제6조 (반복 침해자)
① 동일 회원 3회 침해 확정 시 영구 이용정지
② 단순 신고·반박 가능 사안 = 카운트 제외

제7조 (투명성)
① 신고 처리 통계 연 1회 공개 (전기통신사업법 §22-5 의무자 한정)
```

## 적용 조건 (서비스별 분기)
- **Cre8orClub** — UGC 플랫폼 → 신고 채널·임시조치 SOP 표준
- **KISAS** — 학습자료 UGC 한정 → 강사·운영진 검수 후 게시 (사후 신고 사례 적음)
- **§22-5 임계 도달 (매출 10억 또는 DAU 10만 초과 — OR 조건)** → 신고기능·검색제한·게재제한·사전경고·로그기록·성능평가 + 투명성 보고 추가
