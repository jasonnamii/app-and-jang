# d1-ugc-license 템플릿 (UGC 라이선스 조항) — v1.0

## 법적 근거
- **저작권법 §2·§4** (저작물·저작자 정의)
- **저작권법 §11~17** (저작인격권·저작재산권)
- **저작권법 §103** (Notice-and-Takedown)
- **약관규제법 §6·§7** (불공정 조항 무효)

## 필수 구성
1. **IP 귀속 명확화** — 원칙적 사용자 보유. 회사는 "사용 라이선스"만 부여받음 (양도 ✗)
2. **라이선스 범위** — 비독점·전세계·로열티 프리·서비스 운영 필요 범위
3. **2차 사용 별도** — 광고·홍보·외부 협업 시 추가 동의 필요
4. **삭제 시 라이선스 종료** — 사용자 삭제 시 회사 사용권 동시 종료 (다만 백업·로그 보존은 명시 예외)
5. **저작인격권 행사 제한** — 회사 서비스 통상 운영에 한해 동일성유지권 행사 제한 합의
6. **제3자 권리 침해 책임** — 업로더 본인 부담 명시 + 회사 면책

## 문구 예시 (v2.3 — KISAS 유의 IP귀속 홀딩스 케이스 분기)

```
[UGC 라이선스 — Cre8orClub v1.0]

제○조 (콘텐츠 권리)
① 회원이 게시한 콘텐츠의 저작권은 회원에게 귀속합니다.
② 회원은 회사에 대하여 다음 범위 내에서 무상·비독점·전세계
   사용 라이선스를 부여합니다.
   1. 서비스 내 게시·전송·저장·복제
   2. 검색·추천·통계 처리
   3. 서비스 운영에 필요한 형식 변환
③ 회사가 콘텐츠를 광고·홍보·외부 제휴 목적으로 사용하려면
   회원의 별도 동의를 받아야 합니다.
④ 회원이 콘텐츠를 삭제한 경우 ②항의 라이선스는 즉시 종료됩니다.
   다만 법령 보존 의무에 따른 백업·로그는 보유 기간 내 예외입니다.
⑤ 회원은 본인이 게시한 콘텐츠에 대한 모든 권리를 보유함을 보장하며,
   제3자 권리 침해 시 회원이 직접 책임을 부담합니다.

제○조 (제3자 권리 침해)
회사는 권리침해 신고 접수 시 저작권법 제103조 및 정보통신망법
제44조의2에 따라 지체없이 임시조치할 수 있으며, 게시자에게
통지하여 이의제기 절차를 제공합니다.
```

## 적용 조건 (서비스별 분기)
- **Cre8orClub (디폴트)** — 사용자 IP 귀속 + 회사 사용 라이선스 모델 (표준)
- **KISAS (홀딩스 IP 귀속 케이스)** — 학습자료는 회사 IP 귀속 약정 가능 (학원 표준 UGC 템플릿 적용 ✗) — **KISAS 특수 맹점 3건**(절대규칙 #16) 자동 적용
- **AI 학습 사용** — 별도 동의 필수 (NYT v. OpenAI 분쟁 후속 + AI 기본법 §27)

## v2.3 신규 주의
