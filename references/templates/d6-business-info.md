# d6-business-info 템플릿 (사업자 정보 고지) — v1.0

## 법적 근거
- **전자상거래법 §10·§13** (거래기록 보존·신원 정보 제공)
- **표시광고법 §3·§4** (부당한 표시·광고 금지)
- **부가가치세법 §8** (사업자등록)
- **통신판매업 신고** (전상법 §12 — 별표 1 기준 매출 도달 시)

## 필수 기재 사항 (전상법 §13)
1. 상호·대표자 성명
2. 영업소 주소 (실주소·우편물 수령 가능)
3. 전화번호
4. 이메일 주소
5. 사업자등록번호
6. 통신판매업 신고번호 (해당 시)
7. 호스팅 사업자 (해당 시)
8. 이용약관·개인정보처리방침 링크

사업자등록번호      : XXX-XX-XXXXX  [사업자정보 조회]

## 표시 위치
- **앱**: 설정 > 사업자 정보 (Always 접근 가능)
- **웹**: 모든 페이지 푸터 + 결제·구매 화면 하단 고정
- **결제 직전 화면**: 통신판매업 신고번호 + 청약철회 안내 동시 표시 (전상법 §13 ②)

## 적용 조건 (서비스별 분기)
- **Cre8orClub** — 수익화 시 통신판매업 신고 필수 (전상법 §12)
- **KISAS** — 학원법 등록 별도 + 사업자등록·통신판매업
- **B2B 한정** — 통신판매업 신고 면제 가능 (B2B 거래 명확화 시)
