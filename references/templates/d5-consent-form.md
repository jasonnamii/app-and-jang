# d5-consent-form 템플릿 (개인정보 수집·이용 동의서) — v1.0

## 법적 근거
- **개인정보보호법 §15·§17·§18·§22·§22-2·§23** (수집·제공·동의)
- **정보통신망법 §50** (마케팅 별건 — d5-marketing-consent)
- **청소년보호법 §16** (만 14세 미만 보호자 동의)

## 필수 구성 (5+1요소)
1. **수집 항목** (필수·선택 구분 명시)
2. **수집 목적** (각 항목별)
3. **보유·이용 기간** (목적 달성 시 즉시 파기 원칙)
4. **거부권 + 거부 시 불이익**
5. **민감정보·고유식별정보** = 별도 동의 (§23·§24)
6. **만 14세 미만 = 법정대리인 동의 + 확인 수단** (§22-2)

## 문구 예시 (v2.3 — 필수·선택 분리)

```
[개인정보 수집·이용 동의서 — Cre8orClub v1.0]

▶ 필수 항목 (서비스 이용 필수)
   1. 수집 항목: 이메일·휴대전화·닉네임·생년월일
   2. 수집 목적: 회원 식별·로그인·서비스 운영·불법행위 대응
   3. 보유 기간: 회원 탈퇴 시까지 (단 통신비밀보호법 등 법령 보존 의무 별도)
   □ 동의함  □ 동의하지 않음 (체크 시 회원가입 불가)

▶ 선택 항목 (선택 동의 — 거부 가능)
   1. 수집 항목: 관심분야·선호 아티스트
   2. 수집 목적: 맞춤 추천·통계
   3. 보유 기간: 회원 탈퇴 또는 동의 철회 시까지
   □ 동의함  □ 동의하지 않음
   ※ 거부 시 일부 추천 기능 제한

▶ 만 14세 미만 가입 ✗ — 가입 차단

▶ 민감정보 수집 (해당 시 별도 화면)
   1. 항목: 건강정보·종교·정치성향·생체정보 등
   2. 별도 동의 필수 (§23)
   □ 동의함  □ 동의하지 않음

▶ 고유식별정보 수집 (본인확인 시)
   1. 주민등록번호 = 법령 근거 없이 수집 ✗ (§24-2)
   2. 외국인등록번호·여권번호·운전면허번호 = 별도 동의 + 암호화
```

## 적용 조건 (서비스별 분기)
- **Cre8orClub** — 만 15세 이상 정책 → §22-2 회피
- **KISAS** — 학생 다수 미성년 → §22-2 법정대리인 동의 플로우 필수 (휴대전화 본인인증·신용카드 인증 등 4수단 중 택1)
- **글로벌** — GDPR Art.6·7 (자유로운·특정·정보기반·명시적) + COPPA VPC + PIPL 별도 동의
